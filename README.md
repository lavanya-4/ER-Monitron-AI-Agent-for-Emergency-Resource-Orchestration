# 🚑 ER Monitron – AI Emergency Dispatch System

## 📌 Overview  
ER Monitron is an **AI-powered emergency response system** that optimizes hospital dispatch decisions using **patient vitals, hospital resources, and real-time traffic data**. Built during the **NVIDIA Hackathon (Mar 2026)**.
 deployed url : https://v0-er-nvidia-hackathon.vercel.app/

## 🎯 Problem  
Emergency routing often lacks real-time insight into **hospital capacity, traffic, and patient risk**, leading to delays and suboptimal decisions.

---

## ⚙️ Approach  
Implemented a **multi-step decision pipeline**:
- Patient triage and vitals extraction  
- Hospital resource retrieval  
- Traffic and delay estimation  
- Clinical risk modeling  
- **Weighted scoring algorithm** to select the optimal hospital  

---

## 🤖 AI Components  
- **NVIDIA Llama-3.1 Nemotron-70B** – reasoning and dispatch planning  
- **Nemotron-3-Nano** – fast classification (crisis detection)  
- Hybrid **rule-based + AI decision system**

---

## 🧰 Tech Stack  
React, Vercel, NVIDIA NeMo/Nemotron, APIs
